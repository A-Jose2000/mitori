# Mitori

Mitori is a read-only PostgreSQL visualizer for VS Code.

It helps developers see schemas, tables, columns, reconstructed SQL column definitions, keys, relationships, and sample rows without leaving the editor.

## Usage

1. Open a PostgreSQL project in VS Code.
2. Add `DATABASE_URL` to `.env`.
3. Open the Mitori sidebar.
4. Connect to the database.
5. Inspect tables and preview rows.
